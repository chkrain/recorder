# Автоматический Screen Recorder

Автономный скрипт для записи экрана с гибкими настройками.

## Быстрый старт

```bash
# Сделать скрипты исполняемыми
chmod +x *.sh

# Запустить установку
./install.sh

# Управление
./manager.sh

# Рекомендуется добавить автозапуск
# Для настройки времени автоудаления изменить в screen_recorder.sh RETENTION_HOURS=48 (где 48 - часы), настройка фпс (рекомендуется не более 10) FPS=5 (где 5 - количество фпс), для настройки времени записи DURATION_MINUTES=1 (где 1 - количество минут на каждое видео). CRF - режим кодирования, чем меньше значение, тем выше качество и размер файла, рекомендуется 23-28 в зависимости от возможностей хранения и использования памяти.
# Видео сохраняются в /home/user/screen_recordings, название создаваемых папок ЧИСЛО/МЕСЯЦ/ГОД-ЧАС, название видео record_минута/секунда_начала_записи.
